🔥🅿️ Parking Hell 🅿️🔥

Hi Chat. Hell-p me. I.want.to.park.

Parking Hell is a simple LLM-powered parking assistant that helps drivers find nearby parking options based on destination, vehicle type, walking distance, parking duration, and preference.

Background

This project was inspired by a funny but relatable frustration shared by @TimmyTubbyTV and @supercatkei: maps apps can guide users to a destination, but they do not really help users decide where to park.

Simple Flow

User destination
      ↓
Get latitude and longitude
      ↓
Find nearby parking
      ↓
Rank parking options
      ↓
Calculate Parking Hell Score
      ↓
Gemini gives recommendation

Features

  • Streamlit chatbot UI
  • Gemini API response generation
  • Nominatim geocoding
  • Overpass API parking search
  • Parking Hell Score
  • Simple prompt-injection guardrail
  • Light Singlish-style chatbot personality

How to Run

pip install -r requirements.txt
streamlit run app.py
